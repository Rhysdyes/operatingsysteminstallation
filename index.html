Operating System Installation Tutorial

Choose the method that suits your setup.

Method 1: Using a Bootable USB Drive

  1. First, make sure that your system supports the x32bit or x64bit operating system.
  2. After that, go to the microsoft website and download the windows file or by clicking the link below.
  3. Download Windows 11
  4. Download Windows 10
  5. Insert the USB drive into your computer.
  6. select upgrade if you want to update your windows into the latest version, or if you're just installing the operating for the first time, select the option below.
  7. You can restart your computer if you're done installing the file in your USB.and enter BIOS/UEFI settings (usually by pressing F2, F12, or Del during boot).
  8. Set the USB drive as the primary boot device.
  9. Save changes, restart, and follow the on-screen instructions to install the OS.
